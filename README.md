# VibeLens — Camera Movement Learning App

A self-paced web app that teaches casual moviegoers how to identify camera movements in film.

## Team Members
- Laura Avila — Laura022
- Jessie Li — jessie-qs-li

## How to Run (Final Version)
1. Clone the repository
2. Run `python -m http.server 8000`
3. Open `http://localhost:8000/ui_kits/lesson-app/index.html`

## Old Flask Prototype (HW10)
1. Run `python app.py`
2. Open `http://127.0.0.1:5000`
